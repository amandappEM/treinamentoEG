# Treinamento Pré Envio Gráfico (Pré-EG)

Treinamento interativo em HTML para consultores de Sucesso da **Estante Mágica**, com foco no processo de Pré Envio Gráfico — os 7 dias úteis que antecedem o envio dos livros para produção.

## 🎯 Sobre

O Pré-EG é a etapa final antes dos livros entrarem em produção. É o momento de garantir que cada obra dos alunos chegue perfeita ao Evento de Autógrafos — sem refação, no prazo, com os dados corretos.

Este treinamento reúne em um site único:

- Visão geral do processo
- Checklist interativo (interna + escola)
- Cadência detalhada D-7 até D0, com fluxograma
- Discurso completo, em abas por cenário
- Rotina, processos e pontos de atenção
- Pontos críticos, objeções e como agir em cada caso
- Modelos de e-mail (Tentativa 1, 2 e 3) com botão de copiar
- Templates do WhatsApp/BLIP com busca e cópia rápida

## ⭐ Ferramentas de fixação

- **Flashcards** — 8 conceitos-chave para revisar antes de fechar
- **Simulador de cenários** — decisões reais com feedback imediato
- **Quiz final** — 8 perguntas com correção e pontuação

## 🚀 Como usar

Basta abrir o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Safari, Edge). Funciona offline, não depende de servidor.

Navegação:

- Menu lateral à esquerda
- Botões **Anterior / Próximo** no rodapé de cada módulo
- Atalhos de teclado: **←** e **→** para navegar entre módulos

Consultores de Sucesso da Estante Mágica com escolas a 7 dias úteis do Envio Gráfico. Funciona tanto para **onboarding** de novos consultores quanto para **reciclagem** do time.
